# Wrdli
A CLI-based set of games with daily challenges synced from the original web games.

## How to Play
Run the program from your terminal:

```bash
node wrdli/wrdli.js
```
or
```bash
node cli-connections/cli-connections.js
```
Wrdli:
You have five attempts to guess a secret five-letter word.

After each guess:
- A correct letter in the correct position is highlighted green.
- A correct letter in the wrong position is highlighted yellow.
- A letter not in the word is shown in white.

Type your five-letter guess and press Enter. The game ends when you guess the word or exhaust all five attempts.

### In-game Commands
- `help` — show command help
- `guesses` — reprint your previous guesses
- `exit` / `quit` — end the game

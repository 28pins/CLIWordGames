# Wrdli
A CLI-based Wordle game that uses the same daily word as the original.

> **⚠️ Project Status:** This project is currently in the planning phase. The implementation has not yet been started. Contributions are welcome!

## Overview
Wrdli is a terminal-based clone of the popular Wordle game. The goal is to create a faithful recreation that synchronizes with the original Wordle's daily word, allowing players to enjoy the same puzzle from the command line.

## Planned Features
- Daily word synchronization with the original Wordle
- Six attempts to guess the five-letter word
- Color-coded feedback (green/yellow/gray) for each guess
- Terminal-based interface
- Future: Connections game support (mentioned in roadmap)

## How to Play
Once implemented, you will be able to run the program from your terminal. You'll have six attempts to guess a secret five-letter word.

After each guess:
- A correct letter in the correct position is highlighted **green**
- A correct letter in the wrong position is highlighted **yellow**
- A letter not in the word is shown in **gray**

Type your five-letter guess and press Enter. The game ends when you guess the word or exhaust all six attempts.
